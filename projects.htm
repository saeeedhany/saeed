Projects

A collection of projects I've built over the years — from command-line tools to web applications, each reflecting my journey in understanding systems, improving workflows, and exploring new technologies.

CLI Tools

Terminal-based tools designed to simplify complex workflows and remove unnecessary friction from everyday development tasks.

deasy (2026)

A lightweight command-line wrapper for yt-dlp that removes the complexity of long flags and confusing options.

Built with: Bash

Features: Simplified download interface, preset configurations, batch operations, automatic format selection.

yduts (2025)

A minimalist study timer built for the terminal with zero distractions and maximum clarity.

Built with: Bash

Features: Pomodoro technique support, customizable intervals, terminal notifications, session tracking.

Unity CLI Manager (2025)

Command-line tool for creating, organizing, and managing Unity projects without Unity Hub.

Built with: Bash

Features: Project scaffolding, version management, template support, batch project operations.
